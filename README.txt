1. Содержимое пакета:

 - API для использования сервиса отправки сообщений sms16.ru
 - Пример реализации на Ruby


2. Запуск примеров:

Для запуска примеров необходимо создать MySQL базу данных, структура которой приведена в db.sql
По умолчанию база заполняется данными несуществующих пользователей и номеров, которые необходимо заменить на реальные.


3. Описание API:

Класс SmsApi содержит следующие функции:

# функция получения баланса пользователя
getBalance(login, password)
  login - логин пользователя в системе
  password - пароль пользователя в системе 

# функция отправки сообщения
sendMessage(login, password, type, sender, text, recs, vcard)
  login - логин пользователя в системе
  password - пароль пользователя в системе 
  type - Тип отправляемого сообщения (sms, flashsms, wappush, vcard)
  sender - Имя отправителя
  text - Текст сообщения
  recs - Список получателей
  vcard - Данные пользователя для визитной карточки

# функция проверки доступных отправителей
getSenders(login, password)
  login - логин пользователя в системе
  password - пароль пользователя в системе 

# функция получения состояния сообщений
getStates(login, password, smsIds)
  login - логин пользователя в системе
  password - пароль пользователя в системе 
  smsIds - список id отправленных смс

# функция получения входящих сообщений 
getIncomingMessages(login, password, start, end)
  login - логин пользователя в системе
  password - пароль пользователя в системе 
  start - начальная дата проверки входящих смс в формате YYYY-MM-DD HH:MM:SS
  где, YYYY-год, MM-месяц, DD-день, HH-часы, MM-минуты, SS-секунды.
  end - конечная дата проверки входящих смс

4. Использование:

Для использования gem в Вашем скрипте необходимо выполнить следующие действия.

Установка gem:
	gem install SmsAPI

Использование gem:
	require 'SmsAPI'
